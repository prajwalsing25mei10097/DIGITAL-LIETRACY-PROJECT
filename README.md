About the Project

The main aim of this project was to understand how students can use digital tools in a proper and responsible way. It’s not only about knowing how to use apps or websites, but also about things like communication, online safety, and maintaining a good online presence.

The project is divided into 5 different tasks, and each one focuses on a different part of digital literacy.

Project Structure

This is how I’ve arranged everything in the repository:

task-1-presentation/ → Infographic on digital literacy
task-2-portfolio/ → Screenshots of my profiles (GitHub, LinkedIn, etc.)
task-3-platforms/ → Coding platform work and Google Form
task-4-email-etiquette/ → Email drafts and social media checklist
task-5-cybercrime/ → Case study and safety tips
report/ → Final project report
Task Summary

Task 1 – Digital Literacy Infographic
In this task, I made an infographic to explain what digital literacy actually means and why it is important. I used an online tool to keep it simple and easy to understand. It includes things like safe internet usage, useful tools for students, and how to keep a professional image online.

Task 2 – Student Digital Portfolio
Here, I created and improved my profiles on platforms like GitHub and LinkedIn. The idea was to start building a professional identity. I also understood how these platforms can be helpful for internships and future opportunities.

Task 3 – Platforms Exploration
This task had two parts:

Solving a basic problem on a coding platform
Creating a Google Form as a quiz

This helped me understand how coding platforms work and also how to use tools for creating and sharing forms.

Task 4 – Email Etiquette
In this task, I wrote two formal emails:

One for requesting an assignment extension
One for applying to an internship

I also made a checklist of do’s and don’ts for social media. This task showed me how important it is to maintain proper tone and clarity while communicating.

Task 5 – Cybercrime Awareness
For the last task, I studied a cybercrime case and created a prevention checklist. It mainly focuses on how such issues happen and what steps we can take to avoid them, especially as students.

What I Learned

From this project, I learned a few important things:

How to present myself better online
How to write proper emails
Basic knowledge about cyber safety
How different platforms can help in learning and growth

Overall, it was a useful project and helped me understand practical use of digital tools.
